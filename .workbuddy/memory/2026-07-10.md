## 个人主页改版 — Hero 首页与项目展示优化

### 改动背景
用户反馈 Hero 首页太空虚、项目展示不够突出，要求整体设计调整。

### Hero 区块改动
- 头像从单层 88px 圆改为 96px 圆 + 双层旋转装饰环（外环 160px 实线带轨道点 / 内环 128px 虚线，反向旋转）
- 新增研究关键词标签行：政策反馈 / 行为干预 / 跨境治理 / 因果推断 / 计算文本分析
- 新增核心数据统计条（上下边框分隔）：1篇论文 / 4项项目 / 19.8K+数据处理 / 353份问卷
- 统计条在 480px 以下移除分隔符并允许换行

### 项目展示改动
- 精选项目（MGTI）增加编号 01 和 header 结构（编号 + 标题 + 副标题）
- 三个项目卡片完全重构：
  - 每张卡片顶部新增 120px 可视化预览区（mini-bar-chart / mini-doc / mini-scatter），带标签
  - 卡片正文区增加编号（02/03/04）、关键数据指标行（学校数/学生数/问卷数等）
  - 卡片布局改为 flex-column，指标行 margin-top:auto 贴底
- 移除旧的 project-icon SVG 图标
